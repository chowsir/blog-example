Simple Blog

This is an example of a simple blog. You could adapt this for use as a personal blog, for example. Pay attention to how margin, padding, and width interact to produce the "right" spacing. The following article is another good guide of how to structure a blog using HTML 5.

HTML5 Semantics

Each blog is an `article`. An `article` is any unit of content that could be published to another website. Think of it like everything that would be put in an article that could be published in a magazine, a newspaper, or a blog.

HTML5's biggest difference from HTML4 and earlier is that it provides more tags that more accurately represent the different parts of a web page. For example, the "HTML for blog posts" is contained in a `section` tag, and each blog post is contained in an `article` tag. The semantics of HTML5 tags are more precise and intuitive than earlier versions of HTML, which would represent the "HTML for blog posts" as a `div` containing other `div` tags.

Using jQuery

There's also some basic jQuery here. jQuery is a library that allows web pages to be responsive. This blog creates the effect of switching pages by dynamically showing and hiding different HTML content in response to click events on the anchor tags on the top of the page.
